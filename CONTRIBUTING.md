# 🤝 المساهمة في Smart EFIS Global

مرحبًا بك، وشكرًا لرغبتك في المساهمة في هذا المشروع الرائد!

## كيف تساهم؟
1. قم بعمل Fork للمستودع.
2. أنشئ فرع جديد: `git checkout -b feature/اسم-الميزة`
3. نفّذ التعديلات.
4. اختبر التغييرات.
5. أرسل Pull Request مع وصف واضح.

## تعليمات أسلوب الكود
- استخدم `camelCase` في JavaScript و `snake_case` في JSON
- ضع التعليقات المهمة في كل وحدة من وحدات الذكاء الاصطناعي أو SOE
- كل وظيفة جديدة يجب أن تُختبر (unit test)
- احترم البنية النمطية للمجلدات

## مراجعة الكود
- تتم مراجعة الكود خلال 48 ساعة
- التغييرات الأمنية تُراجع بسرعة أكبر
