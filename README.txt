مشروع: Smart EFIS Global
-------------------------

الوصف:
نظام معلومات طيران شراعي ذكي تفاعلي يدعم الترجمة والخرائط وتخطيط الرحلات والإنذارات الجوية، مبني على تقنيات الويب الحديثة وتكامل Firebase و React و Termux scripts.

المكونات:
- واجهة React تفاعلية مع دعم اللغتين (عربية، إنجليزية)
- نظام i18n للترجمة
- Firebase Authentication + Firestore
- تطبيق APK مرتبط بالموقع
- Smart EFIS Login
- تنبيهات ذكية بمراحل زمنية قبل كل رحلة
- Mapbox للخرائط
- Flask API لتوقعات الطقس
- دعم Termux لأتمتة الإضافة والتحديث

الاقتراحات التطويرية:
- دعم تعدد المستخدمين بحسب الفئة
- نظام صلاحيات مرن
- إشعارات فورية عبر البريد والتطبيق والموقع
- وضع الطوارئ: تجميد الموقع آليًا عند الهجوم
- زر مبدل اللغة داخل الواجهة
- دعم الترجمة الصوتية لاحقًا
- Auto deploy وتصحيح تلقائي للأخطاء
- تحسين SEO وملفات robots/sitemap

المجلدات:
- src/components/LanguageSwitcher.jsx
- src/App.jsx: مدمج مع useTranslation و FlightDashboard
- src/main.jsx: مستورد فيه i18n.js
- i18n.js: ملف تهيئة الترجمة
