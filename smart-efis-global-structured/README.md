# Smart EFIS Global

Smart EFIS is a quantum-enhanced electronic flight information system with:
- AI-powered threat detection
- Multilingual interfaces (EN/AR)
- Firebase + Netlify dual deployment
- Termux-ready scripts for mobile dev

## Scripts
- `npm run dev` — local dev server
- `npm run build` — production build
- `npm run deploy` — Firebase + Netlify
- `npm run clean` — clean/update environment

## Environment
Use `.env.example` to create your `.env.local` file.
